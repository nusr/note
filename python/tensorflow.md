# tensorflow

## 安装

- 安装python3.6.x,配置环境变量
- 只支持64bit python解释器
- 安装tensorfow `pip3 install --upgrade tensorflow`

## windows 下载源
```
//C:\Users\admin\AppData\Roaming\pip\pip.ini
[global]
index-url=http://mirrors.aliyun.com/pypi/simple/
 
[install]
trusted-host=mirrors.aliyun.com
```

## Mac 下载源

```
// ~/.pip/pip.conf
[global]
index-url=http://mirrors.aliyun.com/pypi/simple/
[install]
trusted-host=mirrors.aliyun.com
```
